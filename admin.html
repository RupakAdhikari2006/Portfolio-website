How it works

All modifications you perform in this panel (editing, adding, or deleting) will be saved to your browser's local storage. This allows you to immediately click around pages on your local site to review updates.

To make these updates public on your live site, go to the Sync & Settings tab. You can download the new configuration files and overwrite the files in your repo folder, or input a GitHub token to commit changes directly.

Option A: Manual File Sync

Download the modified config files to your local drive, then drag and drop or commit them to replace the existing JSON files in your project directory.

Option B: GitHub Direct API Sync

Enter a GitHub Personal Access Token to write directly to your repo files. Changes sync to GitHub Pages without leaving the browser.

🔒 Local Storage Encryption
Your PAT token is stored strictly in your local browser sandbox. It is transmitted directly to the api.github.com servers over SSL, never to any intermediary.
